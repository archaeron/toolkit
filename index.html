Getting Started

Getting started is easy! Our current platform of choice is Node.JS. Although this isn't exclusive our views are shared by a lot of people so we have desired to target node.js as our primary platform. JS Toolkit works in every Node.JS version as it doesn't rely on the platform itself but more the underlying virtual machine (Google's V8).

npm install toolkit -g

This will automatically install JS Toolkit along with any dependencies it requires (currently none). The -g flag allows you to then use toolkit in any project run by node on that machine.

require('toolkit');

var arr = [1,2,3,4,5].shuffle();

There we go! You're now all set and ready to go! Next go checkout the documentation for the latest release (1.2.x) here.
