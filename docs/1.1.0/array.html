Array

  • array Range (start, stop, step)

    Constructs an array containing a number of integers specified by the parameters

    View Example{% highlight js %} Array.range(5, 12, 2) // returns [5,7,9,11] Array.range(5, 8) // returns [5,6,7,8] {% endhighlight %}
    View Code{% highlight js %}function (start, stop, step) { if (stop === undefined) { stop = start; start = 1; } var arr = []; for (var i = start; i <= stop; i += (step || 1)) { arr.push(i); } return arr; }{% endhighlight %}

Array.prototype

  • self Swap (index1, index2)

    Swaps two items within the array given their indexes. This is used internally for Array.prototype.shuffle.

    View Example{% highlight js %} var arr = [5, 10, 15]; arr.swap(0,2); arr = [15, 10, 5] {% endhighlight %}
    View Code{% highlight js %}function (index1, index2) { var value = this[index1]; this[index1] = this[index2]; this[index2] = value; return this; }{% endhighlight %}

  • bool Contains (*values)

    Checks whether the array obtains a certain value. This uses the internal indexOf() method which enforces strict equality (===).

    View Example{% highlight js %} var arr = [5, 10, 15]; arr.swap(0,2); arr = [15, 10, 5] {% endhighlight %}
    View Code{% highlight js %}function (values) { args = Object.values(arguments); return args.every(function(arg) { return !! ~ this.indexOf(arg); }.bind(this)); }{% endhighlight %}

  • self Remove (*values)

    Removes all instances of the given values from the array. Remove uses the indexOf method which enforces strict equality (===).

    View Example{% highlight js %} var arr = [5, 2, 5, 7]; arr.remove(5).remove(2); // [7] {% endhighlight %}
    View Code{% highlight js %}function () { args = Object.values(arguments); args.forEach(function(arg) { if (this.contains(arg)) this.splice(this.indexOf(arg), 1); }.bind(this)); return this; }{% endhighlight %}

  • self Shuffle

    Re-orders all values within the array into a random order. The method is simple and relies on Math.random() to generate random numbers which is automatically seeded.

    View Example{% highlight js %} var arr = this.clone(); for (var index = 0; index < arr.length - 1; index++) { arr.swap(index, Math.round(Math.random(index, arr.length - 1))); } return arr; {% endhighlight %}
    View Code{% highlight js %}function () { var arr = this.clone(); for (var index = 0; index < arr.length - 1; index++) { arr.swap(index, Math.round(Math.random(index, arr.length - 1))); } return arr; }{% endhighlight %}

  • array Clone

    Clones the array by copying all the enumerable values into a new one. Any non-enumerable properties you've defined using Object.defineProperties or alike will be lost. If you don't want that, use Object.clone() instead.

    View Example{% highlight js %} var a = [1,2,3], b = a.clone(); a == b // false a.join(',') == b.join(',') // true {% endhighlight %}
    View Code{% highlight js %}function () { return this.slice(); }{% endhighlight %}

  • array Intersect (*values)

    Finds the intersection between one or more different arrays. This can be useful so to use the array as a set. Equivalent to the ampersand (&) operator. Equality checks are strict - again use the indexOf method.

    View Example{% highlight js %} [1,2,3].intersect([2,3,4], [3,4,5]); // returns [3] {% endhighlight %}
    View Code{% highlight js %}function () { var args = Object.values(arguments); return this.filter(function(val) { return args.every(function(arg) { return arg.contains(val); }); }); }{% endhighlight %}

  • array Diff (*arrays)

    Finds values which are unique to only one array from all that are given as parameters (including the current instance). Regarded as the opposite as intersect. Again equality tests are strict.

    View Example{% highlight js %} [1,2,3].diff([2,3,4], [3,4,5]); // returns [1,5] {% endhighlight %}
    View Code{% highlight js %}function () { var vals = Object.values(arguments).concat([this]).flatten(1); return vals.filter(function(a) { return vals.indexOf(a) == vals.lastIndexOf(a); }); }{% endhighlight %}

  • array Chunk (size)

    Takes an array and slices it into an array of smaller chunks. Very useful for dealing with groups of items at a time.

    View Example{% highlight js %} [1,2,3,4,5,6].chunk(2); // returns [[1,2],[3,4],[5,6]] {% endhighlight %}
    View Code{% highlight js %}function (size) { for (var arr = []; this.length > 0; arr.push(this.splice(0, size))); return arr; }{% endhighlight %}

  • array Unique

    Returns a list of unique items within the array. i.e. If there are are 2 identical values, one will be removed. This again uses indexOf() which performs strict equality checks.

    View Example{% highlight js %} [1,2,2,3,4,3].unqiue() // returns [1,2,3,4] {% endhighlight %}
    View Code{% highlight js %}function () { return this.filter(function(a, idx) { return this.indexOf(a) == idx; }.bind(this)); }{% endhighlight %}

  • mixed Each

    Unlike forEach(), each() allows the loop to be broken and a value returned. The callback is called for each item and given (value, index, self) as its parameters. If it returns a value other than undefined the loop will be stopped and that value returned.

    View Example{% highlight js %} var a = [1,2,3,4,5,6].each(function(val, idx) { if (val == 2) return idx; }); // a = 1; {% endhighlight %}
    View Code{% highlight js %}function (callback) { var idx, result; for (idx in this) if ((result = callback.call(null, this[idx], idx, this)) !== undefined) return result; }{% endhighlight %}

  • array Flatten (level)

    Takes a multi-dimensional array and merges it upwards. Turning it into a single-dimension array. The level parameter specifies how many levels to go down within the array.

    View Example{% highlight js %} [[1,2],[[3]],4,5].flatten() // returns [1,2,3,4,5] {% endhighlight %}
    View Code{% highlight js %}function (level) { if (level === undefined) level = -1; return this.reduce(function(a,b) { return a.concat((Array.isArray(b) && level != 0) ? b.flatten(level - 1) : [b]); }, []); }{% endhighlight %}

  • int Sum

    Like it sounds, simply add all the integers contained within the array up together.

    View Example{% highlight js %} [1,'2',3].sum(); // returns 6 {% endhighlight %}
    View Code{% highlight js %}function () { return this.reduce(function(a, b) { return Number(a) + Number(b); }); }{% endhighlight %}

  • int Product

    Same as sum, except finds the product of all values within the array. ie. a*b.

    View Example{% highlight js %} [2,'2',3].product(); // returns 12 {% endhighlight %}
    View Code{% highlight js %}function () { return this.reduce(function(a, b) { return Number(a) * Number(b); }); }{% endhighlight %}

  • array First (num)

    Returns the first n-number of items within an array. If no parameter is given - this defaults to 1.

    View Example{% highlight js %} [1,2,3].first(2); // [1,2] {% endhighlight %}
    View Code{% highlight js %}function (num) { return this.slice(0, num || 1); }{% endhighlight %}

  • array Last (num)

    Returns the last n-number of items within an array. If no parameter is given - this defaults to 1.

    View Example{% highlight js %} [1,2,3].last(2); // returns [2,3] {% endhighlight %}
    View Code{% highlight js %}function (num) { return this.slice(this.length - (num || 1)); }{% endhighlight %}

  • array Clean

    Removes all falsey values from the array. These can be either; NaN,undefined,null,0 or false

    View Example{% highlight js %} [1,null,2,0].clean // returns [2,3] {% endhighlight %}
    View Code{% highlight js %}function () { return this.filter(function(val) { return !! val; }); }{% endhighlight %}
