String

  • string UUID

    Creates a RFC 4122 compliant UUID. This implementation relies on Math.random(). Even so you can be assured that each generation is pretty much always going to be different.

    View Example{% highlight js %} String.uuid() // returns something like '89996AE3-6FBB-428299C78-670C095256631' {% endhighlight %}
    View Code{% highlight js %}function () { return Array.range(0, 35).map(function(i) { switch (i) { case 8: case 13: case 23: return '-'; case 14: i = 4; break; case 19: i = (Math.random(0, 16).floor() & 0x3) | 0x8; break; default: i = Math.random(0, 16).floor(); break; } return '0123456789ABCDEF'[i]; }).join(''); }{% endhighlight %}

  • int Count (substr, [modifiers='gi'])

    Counts the number of instances of a given sub string. You can also give some RegEx modifiers to determine what to look for. By default this operation is case-insensitive.

    View Example{% highlight js %} 'aBab'.count('ab') // returns 2 'aBab'.count('ab', 'g') // returns 1 {% endhighlight %}
    View Code{% highlight js %}function (substr, mod) { return this.match(RegExp(substr, mod || 'gi')).length; }{% endhighlight %}

  • string Insert (substr, [position=0])

    Inserts a substring at a given position within the string. Position defaults to 0 which will insert the string onto the beginning (opposite of concat).

    View Example{% highlight js %} 'ab'.insert('ab') // returns 'abab' 'ab'.insert('ab', 1) // returns 'aabb' {% endhighlight %}
    View Code{% highlight js %}function (substr, pos) { return this.substr(0, pos || 0) + substr + this.substr(pos || 0); }{% endhighlight %}

  • string Swapcase

    Toggle's the case of each character within the string. Uppercase -> Lowercase, Lowercase -> Uppercase.

    View Example{% highlight js %} 'aBc'.swapcase() // returns 'AbC' {% endhighlight %}
    View Code{% highlight js %}function () { return this.chars().map(function(a) { return /[a-z]/.test(a) ? a.toUpperCase() : a.toLowerCase(); }).join(''); }{% endhighlight %}

  • string Rpad (length, chars)

    Continues to add a set of characters to the end of the string until it reaches a certain length.

    View Example{% highlight js %} 'a'.rpad(5) // returns 'a ' 'a'.rpad(5, '123') // returns 'a1231' {% endhighlight %}
    View Code{% highlight js %}function (len, chars) { for (var str = this; str.length < len; str += (chars || ' ')); return str.substr(0, len); }{% endhighlight %}

  • string Lpad (length, chars)

    Continues to add a set of characters to the beginning of the string until it reaches a certain length.

    View Example{% highlight js %} 'a'.lpad(5) // returns ' a' 'a'.rpad(5, '123') // returns '1321a' {% endhighlight %}
    View Code{% highlight js %}function (len, chars) { chars = (chars || ' ').reverse(); for (var str = this; str.length < len; str = (chars + str)); return str.substr(str.length - len); }{% endhighlight %}

  • string Soundex

  • string Distance (string)

    Calculates the distance between 2 strings using Levenshtein's algorithm.

    View Example{% highlight js %} 'aBc'.swapcase() // returns 'AbC' {% endhighlight %}
    View Code{% highlight js %}function (c) { var s, l = (s = this.split("")).length, t = (c = c.split("")).length, i, j, m, n; if(!(l || t)) return Math.max(l, t); for(var a = [], i = l + 1; i; a[--i] = [i]); for(i = t + 1; a[0][--i] = i;); for(i = -1, m = s.length; ++i < m;) for(j = -1, n = c.length; ++j < n;) a[(i *= 1) + 1][(j *= 1) + 1] = Math.min(a[i][j + 1] + 1, a[i + 1][j] + 1, a[i][j] + (s[i] != c[j])); return a[l][t]; }{% endhighlight %}
