Math

  • int Random ([start=0], [end=1])

    Random number between two values.

    View Example{% highlight js %} Math.random(100, 1000) // returns something like 521.61242932 (between 100 and 1000) Math.random() // returns something like 0.61242932 (between 0 and 1) {% endhighlight %}
    View Code{% highlight js %}function (start, end) { return (start = start || 0) + (((end || start + 1) - start) * rand()); }{% endhighlight %}
